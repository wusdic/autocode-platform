# autocode-platform

多项目自动编程平台——基于 Hermes Agent v0.16.0 的完整设计方案与操作手册。

## 包含文件

- **01-最终设计方案.md**：完整架构设计，包含五条底层原则、三层隔离模型、13个角色权限表、需求双层结构、设计委员会多模型质疑机制、先设计再执行的双层硬拦、90轮问题处理、风险规避表。
- **02-从零开始操作手册.md**：13个阶段可逐条执行的操作手册，包含完整的 bash 脚本、Python 插件代码、FastAPI 骨架代码，从零装环境到上生产。

## 技术栈

Hermes Agent v0.16 · Kanban multi-agent · FastAPI · Docker · PostgreSQL · Redis
